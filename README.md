## Instruction
### For development run

```yarn dev ```

### For production built

```yarn build ```

### Note 
In nest.config.json, deployment on github pages

```const nextConfig = {
  reactStrictMode: true,
  swcMinify: true,
  assetPrefix: "/your-repo",
  basePath: "/your-repo",
};
```
For deploying on any other server

```const nextConfig = {
  reactStrictMode: true,
  swcMinify: true,
  assetPrefix: "/, 
  basePath: "/",
};
```
